## Concept

It has been developed as our project n°2 during a 5 monthes formation focus on React. So far there is no backend but new features are coming soon !!

## Setup & Use

### Project Initialization

- Clone this repo, enter it
- If you are using `yarn` or `pnpm`, adapt the `config/cli` in `package.json`
- Run command `npm install`
- Copy .env.example to .env and fill the variables
- Run command `npm run dev`

### Available Commands

- `dev` : starts app in development mode
- `build` : builds the app for production
- `preview` : starts a local server to preview the production build
- `lint` : runs validation tools
- `format` : Formats the code
- `docker:build` : Builds the docker image
